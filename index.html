Configurer la base de données
Colle ici ton bloc firebaseConfig pour synchroniser les données entre tous tes appareils et tes camarades.
1. Va sur console.firebase.google.com
2. Crée un projet → active Firestore Database en mode test
3. Paramètres du projet (⚙️) → onglet Général → section "Vos applications" → icône Web </>
4. Copie le bloc firebaseConfig = { … } et colle-le ci-dessous
✅ Configuration Firebase enregistrée
La configuration est sauvegardée localement dans ton navigateur — tu n'auras à la saisir qu'une seule fois par appareil.
